撤销订单

取消指定的永续合约订单。

接口信息​

  • 方法: DELETE
  • 路径: /fapi/v1/orders/{id}
  • 鉴权: 需要签名认证（详见 公共模块 · 签名说明）

请求参数​

参数位置类型必填说明
idpathstring是订单 ID

请求示例​

DELETE /fapi/v1/orders/1234567890123456789

响应字段​

返回被撤订单的完整信息。

字段类型说明
codeinteger状态码
messagestring返回消息
data.idstring订单 ID
data.symbolstring交易对
data.client_order_idstring客户端订单 ID
data.statusinteger订单状态
data.cancel_reasonstring取消原因
data.pricestring委托价格
data.qtystring委托数量
data.filled_qtystring已成交数量
data.filled_quotestring已成交金额
data.avg_pricestring成交均价
data.total_feestring累计手续费
data.fee_assetstring手续费币种
data.updated_atinteger更新时间（毫秒）
