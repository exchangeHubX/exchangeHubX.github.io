交易对榜单

获取热门榜、涨幅榜、新币榜。

接口信息​

  • 方法: GET
  • 路径: /fapi/v1/symbols/boards

请求参数​

参数位置类型必填说明
limitqueryinteger否每个榜单返回数量，默认 10
tagquerystring否标签：new / meme / innovation

请求示例​

GET /fapi/v1/symbols/boards?limit=20

响应字段​

字段类型说明
codeinteger状态码
messagestring返回消息
data.hotarray热门榜（交易对列表）
data.gainerarray涨幅榜
data.newarray新币榜

每项为完整交易对对象，字段参考"交易对列表"。

响应示例​

{
  "code": 0,
  "message": "success",
  "data": {
    "hot": [
      {
        "symbol": "BTC_USDT",
        "base_asset": "BTC",
        "quote_asset": "USDT",
        "margin_asset": "USDT",
        "contract_type": 1,
        "status": 1,
        "max_leverage": 125
      }
    ],
    "gainer": [],
    "new": []
  }
}
