成交推送

订阅指定交易对的实时成交记录（逐笔成交）。

  • 频道格式：trade@{symbol}
  • 推送频率：实时（无节流）
  • 权限：公开

订阅请求​

{
  "op": "subscribe",
  "args": ["trade@BTC_USDT"]
}

取消订阅​

{
  "op": "unsubscribe",
  "args": ["trade@BTC_USDT"]
}

推送示例​

{
  "ch": "trade@BTC_USDT",
  "d": {
    "id": 123456789,
    "idx": 0,
    "s": "BTC_USDT",
    "p": "49000.50",
    "q": "0.01",
    "qq": "490.005",
    "ts": 1,
    "t": 1768205315166
  }
}

推送字段​

字段类型说明
idnumber撮合 ID
idxnumber成交序号（同一撮合可能产生多笔成交）
sstring交易对
pstring成交价格
qstring成交数量
qqstring成交额
tsnumber吃单方向：1=买入，2=卖出
tnumber成交时间（毫秒）
